### Ürün Açıklaması
Oyunumuz farklı low poly araçlarla yarış mekaniğinin birleştiği, farklı pistlerde gerçekleşen bir araba yarışı oyunudur.
### Ürün Özellikleri
LowPoly araçlar, yapay zekaya karşı yarışmaktadır.
